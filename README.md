# Fraud Detection in Government Schemes

This project detects fraudulent activities in government schemes using Machine Learning and Graph Neural Networks.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- XGBoost
- Isolation Forest
- Graph Neural Networks (GNN)

## Features
- Fraud transaction detection
- Suspicious pattern analysis
- Ensemble prediction system
- Explainable AI using SHAP
